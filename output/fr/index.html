Jug

Jug is a static website toolkit. It transforms Twig templates with hardcoded content into HTML.

It does not load content through Markdown. All content is entered straight into the templates.

On top of the basic Twig to HTML workflow, it also does the following things:

  • compresses/optimizes any images on the site
  • adds a build-specific hash to all assets for easy cache-busting on deploy
  • translates any content that is tagged with the `trans` Twig filter
  • if multiple locales are configured, it generates a version of the site in each locale

Since the templates are built in Twig, you get access to blocks, includes, basic Twig filters and more. There is also a configuration file in basic PHP format, which you have access to in every template.

This is an educational project only intended for personal use.
