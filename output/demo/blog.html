Example: blog

Jug isn't a full-on blogging platform, but it does have a couple of tools that can help you set up a blog.

Utilizing the markdown filter and Twig templates, it becomes incredibly easy to write simple articles or blog posts.

For this example. I've made a blog/ folder. Inside, I'll place an example blog post: my-first-blog-post.twig:

{% extends "post.twig" %} {% block main %} {% apply markdown %} # Your first blog post Here is where your content would go. Written in [Markdown](https://en.wikipedia.org/wiki/Markdown). {% endapply %} {% endblock %}

To render all of my posts, I'll create index.twig, which will be the landing page of the blog.

Jug has a simple helper called dir() that returns a list of filenames in a given folder. You can use this to render a list of links to your blog posts.

{% for file in dir("source/blog") %} {% if file != "index.html" %} <li><a> href="blog/{{ file }}">{{ file }}</a></li> {% endif %} {% endfor %}

This will render a very simple list of files/posts.

There is no support for advanced features like pagination, tags or excerpts. The overhead of supporting those would be a burden on the rest of the tool.
