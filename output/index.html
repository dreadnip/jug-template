Handmade websites with less headaches

Building sites by hand is fun. HTML, CSS and JS are great, and they produce clean, fast and safe websites.

The drawback is the time you need to invest and the flexibility you lose.

Jug is my personal tool to minimise the drawbacks of building by hand.

Head over to the features page to see what it's all about.

You can find more in-depth examples on the demo page.

This is mainly educational project only intended for personal use.
